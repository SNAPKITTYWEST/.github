# .github Repository

This repository contains GitHub configuration files and workflows for the organization.

## Contents

- **Workflows** (`.github/workflows/`) - CI/CD pipelines and automation
- **Issue Templates** (`.github/ISSUE_TEMPLATE/`) - Standardized issue reporting
- **Pull Request Template** (`.github/PULL_REQUEST_TEMPLATE.md`) - PR guidelines
- **Contributing Guide** (`.github/CONTRIBUTING.md`) - Contribution guidelines
- **Code of Conduct** (`.github/CODE_OF_CONDUCT.md`) - Community standards
- **Security Policy** (`.github/SECURITY.md`) - Vulnerability reporting
- **Dependabot Config** (`.github/dependabot.yml`) - Dependency updates
- **Settings** (`.github/settings.yml`) - Repository settings management

## Usage

These files are automatically used by GitHub for repository management. Most configurations apply organization-wide when placed in the `.github` repository.

## Maintenance

Update workflows and templates as needed. Changes take effect immediately on supported repositories.
